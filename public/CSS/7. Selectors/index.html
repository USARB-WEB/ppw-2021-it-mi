Selector for unique element

In HTML: 
        id="uniqueElement"
In CSS: 
        #uniqueElement
Example:
Unique element

Selector for class of elements

In HTML: 
        class="uniqueElement"
In CSS: 
        .classOfElements
Example:
Element 1 of Class
Element 2 of Class
Element 3 of Class

Selector for tag

In HTML: 
        [tag]
In CSS: 
        [tag]
Example: Link 1 Link 2 Link 3 Link 4

Selector for all elements(universal selector)

In HTML: 
        [any tag]
In CSS: 
        *
Example: 
See CSS file and remove comments
